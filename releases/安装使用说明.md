## 快速启动

### 一键启动（推荐）

1. 下载并解压 `backend-v1.1.0.zip`
2. 打开命令行，进入解压后的目录：
```bash
cd backend
```

3. 安装依赖：
```bash
npm install
```

4. 启动服务：
```bash
npm start
```

5. 打开浏览器访问：`http://localhost:8519/desktop.html`

### 默认账户

| 用户名 | 密码 | 角色 | 说明 |
|--------|------|------|------|
| `admin` | `admin123` | 管理员 | 可管理所有用户和项目 |
| `demo` | `123456` | 普通用户 | 用于测试演示 |

## 用户账户说明

### 用户注册

1. 在登录页面点击"注册"标签
2. 输入用户名和密码（密码至少6位）
3. 点击"注册"完成账户创建
4. 新注册用户默认为普通用户角色

### 用户角色

| 角色 | 权限 |
|------|------|
| **admin** (管理员) | 全部权限：日志管理、用户管理、项目管理、角色管理 |
| **user** (普通用户) | 日志管理、项目创建与使用 |

## 后端安装

### 1. 下载后端包

下载 `backend-v1.1.0.zip` 并解压到任意目录。

### 2. 安装依赖

```bash
cd backend
npm install
```

### 3. 启动服务

```bash
# 方式一：npm 脚本
npm start

# 方式二：直接运行
node server.js
```

### 4. 验证服务

打开浏览器访问以下地址验证：

| 地址 | 说明 |
|------|------|
| http://localhost:8519/api/health | 健康检查接口 |
| http://localhost:8519/desktop.html | 电脑端界面 |

### 5. 数据库说明

- 数据库使用 SQLite（sql.js 纯 JavaScript 实现）
- 数据存储在 `backend/construction.db` 文件中
- 首次启动会自动初始化数据库和默认账户

## 电脑端使用

电脑端界面已包含在后端服务中，无需单独部署。

### 启动步骤

1. 确保后端服务已启动
2. 打开浏览器访问：`http://localhost:8519/desktop.html`
3. 使用管理员账户登录：`admin` / `admin123`
4. 或使用演示账户登录：`demo` / `123456`

### 功能说明

| 功能 | 说明 |
|------|------|
| 用户注册 | 登录页面可切换到注册表单 |
| 日志列表 | 查看所有施工日志 |
| 筛选功能 | 总日志 / 本周 / 今日 快速筛选 |
| 新建日志 | 点击底部 "+" 按钮创建新日志 |
| 天气自动获取 | 新建日志时点击"获取当前天气" |
| 数据统计 | 点击"统计"标签查看统计图表 |
| 项目管理 | 创建、管理项目及成员（已激活） |
| 用户管理 | 管理员专属功能（仅 admin 可见） |

### 单独部署电脑端

如需单独使用电脑端界面：

1. 下载 `desktop-v1.1.0.zip`
2. 解压 `desktop.html`
3. 确保后端服务运行在 `http://localhost:8519`
4. 直接用浏览器打开 `desktop.html` 即可使用

## 管理员功能

### 访问管理员功能

1. 使用 `admin` 账户登录
2. 侧边栏会显示"用户管理"菜单
3. 可使用项目管理功能

### 用户管理

管理员可以：

- 查看所有用户列表（显示用户名、角色、加入时间、参与项目数）
- 删除用户（无法删除自己，无法删除最后一个管理员）
- 切换用户角色（admin / user）
- 创建新用户

### 项目管理

管理员和普通用户都可以：

- 创建新项目
- 查看项目列表
- 管理项目成员（添加/移除）
- 删除项目

## 安卓端开发

### 1. 下载源码

从 GitHub 克隆或下载 `mobile-app` 目录：
```bash
git clone https://github.com/useEffectGuy/shigongrizhi.git
cd mobile-app
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置 API 地址

编辑 `src/utils/request.ts`，确认 API 地址：

```typescript
const BASE_URL = 'http://localhost:8519';  // 根据实际部署地址修改
```

### 4. 开发模式

```bash
# 微信小程序（推荐）
npm run dev:weapp

# H5 模式
npm run dev:h5
```

### 5. 构建生产版本

```bash
# 微信小程序
npm run build:weapp

# H5
npm run build:h5
```

### 6. 编译 APK

如需生成 Android APK，有以下方式：

**方式一：使用 Taro 打包**
1. 构建微信小程序：`npm run build:weapp`
2. 使用微信开发者工具打开 `dist` 目录
3. 点击"上传"或使用云开发部署

**方式二：使用 UniApp**
将 Taro 项目转换为 UniApp 后使用 HBuilderX 打包

**方式三：使用 Cordova/React Native**
使用 H5 构建版本，配合 Cordova 或 React Native 封装为原生应用

## 常见问题

### Q1: 端口 8519 被占用怎么办？

**解决方法：**
1. 查看占用进程：
```bash
netstat -ano | findstr :8519
```

2. 结束占用进程：
```bash
taskkill /F /PID <进程ID>
```

### Q2: npm install 很慢或失败？

**解决方法：**
1. 使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

2. 或者使用代理：
```bash
npm install --registry=https://registry.npmmirror.com
```

### Q3: 天气获取失败？

**原因：** 天气功能需要联网访问 Open-Meteo API

**解决方法：**
1. 确保电脑已联网
2. 允许浏览器访问位置信息
3. 如无法联网，可手动输入天气信息

### Q4: 电脑端无法连接后端？

**检查项：**
1. 后端服务是否已启动
2. 防火墙是否阻止了 8519 端口
3. 确认访问地址为 `http://localhost:8519/desktop.html`

### Q5: 如何更改服务端口？

编辑 `backend/server.js`，修改端口配置：

```javascript
const PORT = process.env.PORT || 8519;  // 修改 8519 为你想要的端口
```

然后同步修改：
- 安卓端 `src/utils/request.ts` 中的 API 地址
- 电脑端 `desktop.html` 中的 API 地址

### Q6: 忘记管理员密码怎么办？

**方法一：使用 admin 账户登录后，在用户管理中修改密码**

**方法二：删除数据库重新初始化（会丢失所有数据）：**
1. 删除 `backend/construction.db` 文件
2. 重新启动后端服务
3. 默认账户会重新创建：`admin` / `admin123`

### Q7: 普通用户看不到"用户管理"菜单？

这是正常行为。"用户管理"菜单仅对 admin 角色可见，普通用户无法访问。

### Q8: 如何将普通用户提升为管理员？

1. 使用 `admin` 账户登录
2. 进入"用户管理"页面
3. 找到目标用户，点击"设为管理员"按钮
